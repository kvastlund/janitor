# Scripts
A collection of personal scripts.

## Requirements
Arch Linux base: https://archlinux.org/packages/core/any/base/



## Setup
Run `scripts_to_path.sh` at startup.

## Usage
`list-scripts` - List all scripts in **Scripts** that are available on PATH.
