# Contributing to Janitor
- A line of source code that only consists of a comment should be no more than 80 characters long.
- A comment at the end of a line of source code should be moved before the line in question if the line, with or without the comment, exceeds 80 characters in length.
- A line of output should be no more than 80 characters long, unless technical requirements necessitate it.
